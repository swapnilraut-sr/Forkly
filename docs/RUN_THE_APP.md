# 🍴 Running the Forkly App — Start Here

Welcome! This is a friendly starting point for getting the **Forkly** app running
on your **Mac**, even if you've never written code or used a "Terminal" before.

You do **not** need:
- ❌ a real iPhone or Android phone
- ❌ an Apple account or a GitHub account
- ❌ any coding experience

Everything runs on your Mac inside a **simulated phone** (a phone-shaped window
on your screen that behaves just like a real device).

---

## 📖 Pick your guide

There are two ways to run Forkly. **You only need one** — pick whichever phone
type you'd like to see it on. If you're not sure, **start with iOS** (it's a bit
quicker to set up on a Mac).

| Guide | Runs Forkly as… | Best if… |
|---|---|---|
| 📱 **[Launch on iOS Simulator](./LAUNCH_IOS.md)** | a virtual **iPhone** | You want the quickest path on a Mac ⭐ |
| 🤖 **[Launch on Android Emulator](./LAUNCH_ANDROID.md)** | a virtual **Android phone** | You specifically want to see the Android version |

Each guide is fully self-contained — it lists every tool you need, how to install
each one (with copy-paste commands), and walks you all the way to the app opening
on screen. Every step also tells you how to check if something is **already
installed**, so you can safely skip steps you've already done.

---

## ⏱️ What to expect

- **First-time setup takes a while** — mostly waiting for large downloads
  (the developer tools are big). Set aside an hour or two and let downloads run.
- **After the first time, launching is fast** — under a minute or two.
- **It's hard to break anything.** These are all standard, safe developer tools.
  If a command shows a scary-looking error, don't worry — see the tips below.

---

## 🆘 If you get stuck

Every guide ends with an **"If something goes wrong"** section covering the most
common hiccups. The quickest way to get unstuck:

1. Take a screenshot of the Terminal window showing the error
   (press **Cmd + Shift + 4**, then **Spacebar**, then click the window — it
   saves to your Desktop).
2. Send it to **Swapnil**.

---

## 🤔 Good to know

- **Why not just a simple "download and open" app?** Forkly is a React Native app
  that includes Salesforce's Personalization technology built into the phone code,
  so it has to be *compiled* on your Mac the first time. That's why there are a few
  setup steps rather than a one-click installer.
- **The simulated phone is the real app.** The iOS Simulator and Android Emulator
  run the exact same app you'd get on a physical device — so what you see is a true
  preview.
- **Making changes:** if you (or someone) edit the app's code while it's running,
  it updates on screen automatically when the file is saved — no need to start over.

Happy testing! 🎉
